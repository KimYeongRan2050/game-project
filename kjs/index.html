게임 설명

Reaction Speed Test Game

초록색이 되었을 때 얼마나 빨리 클릭하는지를 측정하는 반응속도 테스트 게임

게임 방법

  • 회색 박스를 클릭하면 게임이 시작
  • 잠시 후 박스가 초록색으로 바뀌면 클릭
  • 빨리 누르면 실패

특징

  • 간단한 조작으로 누구나 쉽게 플레이 가능
  • 클릭 반응속도를 측정하여 초 단위로 결과 표시
  • 최근 기록과 평균 반응속도를 자동으로 저장 및 표시
  • 기록 초기화 버튼으로 언제든지 기록 리셋 가능

개선 방안

  • 순위 기능 추가
  • 색상, 반응 조건을 커스터마이징할 수 있는 기능
